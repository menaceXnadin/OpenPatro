## What It Is

OpenPatro is a lightweight system utility that brings calendar awareness into everyday desktop workflow. Instead of acting like a full web portal, it works as a local desktop-first calendar companion.

## What It Does

- Shows Nepali calendar date context in a desktop-friendly interface.
- Provides quick-access calendar views through popup and tray-style interactions.
- Supports smooth month/day browsing for day-to-day date lookup.
- Includes search-oriented calendar navigation for faster date discovery.
- Keeps calendar data accessible in an app-style experience tailored for Windows users.

## Project Status

This project is in active development, and features or visuals may change over time.
